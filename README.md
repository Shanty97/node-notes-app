## Features

  - CRUD operations
  - Cool labelled alerts

## What is used ?

  - yargs
  - chalk

### Installation

node-notes-app requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and execute the file..

```sh
$ cd node-notes-app
$ npm install
$ node index.js --help
```
